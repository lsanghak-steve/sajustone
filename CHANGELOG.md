# Changelog

## 1.2.0 - 2026-05-27

### 추가

- 상세 만세력 리포트의 대운, 세운, 월운 표에 모바일 핵심 요약 카드 추가
- 대운/세운 현재 행 강조 표시 추가
- 표 영역 좌우 스크롤 안내 문구와 키보드 포커스 스타일 추가

### 변경

- 모바일 표 셀 간격, 줄바꿈, sticky 첫 열 스타일 개선
- 화면과 문서 버전을 `1.2.0`으로 갱신
## 1.1.0 - 2026-05-27

### 추가

- 지장간별 십성 상세 계산을 상세 만세력 리포트에 추가
- 원국 구조 카드 추가
- 도화, 역마, 화개, 장성, 천을귀인, 문창, 양인, 괴강, 공망 주요 신살 계산 추가
- 주요 신살 카드에 적용 위치, 기준, 활용 해석 표시
- `new.md` 진행 상태 기반 다음 기능 개발 관리 방식 적용

### 변경

- 상세 사주 해설에 지장간, 십성, 주요 신살 안내 문단 반영
- 화면과 문서 버전을 `1.1.0`으로 갱신
## 1.0.0 - 2026-05-27

첫 정식 릴리스입니다.

### 주요 기능

- 양력/음력 생년월일, 성별, 출생시간 입력 지원
- 음력 윤달 입력 및 한국천문연구원 API 기반 양력 변환
- 한국천문연구원 음양력 API 기반 공식 음력 정보, 일진, 율리우스 적일 표시
- 공식 일진을 활용한 일주 보정
- 사주 4주와 오행 분포 계산
- 부족 오행 안내
- 올해 운세 카드 제공
- 상세 만세력 리포트 제공
  - 사주 4주
  - 오행 분포
  - 십성/지장간/12운성
  - 대운/세운/月운
  - 천간/지지 관계
  - 신살 요약
- 절기 차이 기반 대운 시작 나이 계산
- 약 2000자 내외 사주 상세 해설 제공
- 부족 오행 기반 천연석 추천
- 원석/팔찌 이미지 표시 및 구매 링크 연결
- 원석 전체 보기 페이지 제공
- Vercel 루트 리다이렉트와 서버리스 API 구성

### 운영 메모

- `.env`는 Git에 포함하지 않습니다.
- Vercel 배포 시 `KASI_SERVICE_KEY`, `KASI_LUNAR_API_ENDPOINT`를 Vercel Environment Variables에 등록해야 합니다.
