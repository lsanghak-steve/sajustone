# 사주 상세 해설 기능 계획

## 기능 목표

생년월일과 출생시간을 입력해 사주 결과를 조회했을 때, 단순히 사주 기둥과 오행 분포만 보여주는 것이 아니라 사용자가 자신의 사주를 이해할 수 있도록 약 2000자 내외의 상세 해설을 함께 제공한다.

## 사용자 흐름

1. 사용자가 이름, 성별, 생년월일, 출생시간을 입력한다.
2. `부족 오행 조회` 버튼을 누른다.
3. 기존 결과인 년주, 월주, 일주, 시주와 오행 그래프가 표시된다.
4. 부족 오행 안내 아래에 `사주 상세 해설` 영역이 추가로 표시된다.
5. 해설은 입력값으로 계산된 사주 기둥, 일간, 강한 오행, 부족한 오행, 출생시간 입력 여부를 반영한다.

## 포함할 해설 내용

- 사주팔자의 기본 의미
- 년주, 월주, 일주, 시주의 역할 설명
- 일간을 기준으로 본 자기 기질 설명
- 오행 분포를 바탕으로 강한 기운과 부족한 기운 해석
- 부족 오행을 보완하는 생활 방향
- 결과를 절대적 운명으로 단정하지 않는 안내 문구
- 현재 계산 방식의 한계 안내: 양력 기준, 고정 절기 날짜, 간단 오행 점수 방식

## 구현 범위

- `index.html`: 결과 영역에 상세 해설 섹션 추가
- `app.js`: 계산 결과를 기반으로 긴 해설 텍스트를 생성하는 함수 추가
- `styles.css`: 긴 텍스트를 읽기 좋게 보여주는 카드 스타일 추가
- `README.md`: 상세 해설 기능 설명 추가

## 제외 범위

- 실제 절기 시각 기반 만세력 계산
- 음력/윤달 변환
- AI API를 통한 실시간 문장 생성
- 개인정보 저장 또는 서버 전송

## 구현 방식

프론트엔드에서 계산된 `pillars`, `scores`, `deficient` 데이터를 이용해 정적 템플릿 기반 해설을 생성한다. 외부 API 없이 동작하므로 로컬 HTML 환경에서도 사용할 수 있다. 추후 더 정확한 명리 해석 엔진이나 OpenAI API를 붙일 경우 `generateDetailedReading` 함수만 교체할 수 있도록 분리한다.

# 재물운/사업운/연애운/종합운 추가 계획

## 기능 목표

생년월일 조회 결과에서 사용자가 가장 궁금해하는 생활형 해석인 재물운, 사업운, 연애운, 종합운을 별도 카드로 제공한다. 기존 사주 기둥, 오행 분포, 부족 오행, 상세 해설은 유지하고, 그 사이에 핵심 운세 요약을 추가해 결과 화면의 활용도를 높인다.

## 사용자 흐름

1. 사용자가 생년월일과 출생시간을 입력한다.
2. `부족 오행 조회` 버튼을 누른다.
3. 기존 오행 결과와 함께 `운세 요약` 영역이 표시된다.
4. 운세 요약에는 재물운, 사업운, 연애운, 종합운 카드가 표시된다.
5. 각 카드는 현재 사주의 일간, 오행 점수, 강한 오행, 부족 오행을 바탕으로 다른 문장을 보여준다.

## 운세별 해석 기준

- 재물운: 일간이 제어하는 오행을 재성 방향으로 보고, 현실 감각과 결과화 능력을 함께 해석한다.
- 사업운: 일간을 제어하는 오행과 월주의 계절감을 참고해 책임, 조직, 추진 방향을 해석한다.
- 연애운: 관계의 균형감, 표현력, 안정감, 부족 오행을 바탕으로 해석한다.
- 종합운: 전체 오행 분포의 균형도와 강약 차이를 바탕으로 현재 사주의 전체 인상을 정리한다.

## 구현 범위

- `index.html`: 결과 영역에 `운세 요약` 카드 섹션 추가
- `app.js`: 재물운, 사업운, 연애운, 종합운 문구 생성 함수 추가
- `styles.css`: 운세 카드 그리드와 배지 스타일 추가
- `README.md`: 운세 요약 기능 설명 추가

## 주의 사항

현재 계산은 간단형 오행 점수 기반이므로 전문 명리 상담처럼 단정하지 않는다. 문구는 “가능성이 있다”, “의식하면 좋다”, “보완하면 안정된다”처럼 참고형으로 작성한다.

# 현재 년도 운세 추가 계획

## 기능 목표

생년월일 기반 기본 운세와 별도로, 현재 년도의 간지를 계산해 올해의 재물운, 사업운, 연애운, 종합운을 보여준다. 사용자가 매년 다시 조회했을 때 그 해의 흐름을 참고할 수 있도록 한다.

## 사용자 흐름

1. 사용자가 생년월일과 출생시간을 입력한다.
2. `부족 오행 조회` 버튼을 누른다.
3. 기존 `운세 요약` 아래에 `올해 운세` 영역이 표시된다.
4. 올해 운세에는 현재 연도, 해당 연도의 간지, 재물운, 사업운, 연애운, 종합운 카드가 표시된다.
5. 브라우저의 현재 연도를 기준으로 자동 계산한다.

## 해석 기준

- 올해 간지의 천간/지지 오행을 계산한다.
- 일간 기준 재물 오행, 사업 오행과 올해 오행이 만나는지를 본다.
- 올해 오행이 부족 오행을 보완하는지, 강한 오행을 더 키우는지 확인한다.
- 연애운은 올해의 화/토 기운과 사용자의 부족 오행을 함께 참고한다.
- 종합운은 올해 오행이 전체 균형에 주는 영향을 요약한다.

## 구현 범위

- `index.html`: `올해 운세` 카드 섹션 추가
- `app.js`: 현재 연도 간지 계산 및 올해 운세 생성 함수 추가
- `README.md`: 현재 년도 운세 기능 설명 추가

## 제외 범위

- 대운, 세운의 정밀 합충형해 계산
- 월별/일별 운세
- 절기 시각 기반 연도 전환 정밀 보정

## 주의 사항

현재 년도 운세는 간단형 오행 흐름 해석이다. 따라서 결과 문구는 단정형이 아니라 참고형으로 제공한다.

# 부족 오행 기반 천연석 추천 기능 계획

## 기능 목표

사주 계산 결과에서 부족한 오행을 확인한 뒤, 해당 기운을 상징적으로 보완할 수 있는 천연석을 추천한다. 사용자는 자신의 부족 오행에 맞는 천연석 이름, 색상, 의미, 착용/활용 팁을 한눈에 확인할 수 있다.

## 사용자 흐름

1. 사용자가 생년월일과 출생시간을 입력한다.
2. `부족 오행 조회` 버튼을 누른다.
3. 부족 오행 결과 아래에 `천연석 추천` 영역이 표시된다.
4. 부족한 오행이 하나면 해당 오행의 추천석 3개를 보여준다.
5. 부족한 오행이 여러 개면 각 오행별 추천석을 모두 보여준다.

## 추천 기준

- 목(木): 성장, 시작, 기획, 회복을 상징하는 초록 계열 천연석
- 화(火): 표현, 열정, 활력, 자신감을 상징하는 붉은 계열 천연석
- 토(土): 안정, 균형, 현실감, 신뢰를 상징하는 노랑/갈색 계열 천연석
- 금(金): 정리, 결단, 집중, 보호를 상징하는 흰색/금속성 계열 천연석
- 수(水): 지혜, 유연함, 감정 정화, 휴식을 상징하는 파랑/검정 계열 천연석

## 구현 범위

- `index.html`: 결과 영역에 `천연석 추천` 섹션 추가
- `app.js`: 부족 오행을 기준으로 천연석 추천 카드 생성
- `styles.css`: 천연석 카드, 색상 원형 스톤 비주얼, 추천 배지 스타일 추가
- `README.md`: 천연석 추천 기능 설명 추가

## 주의 사항

천연석 추천은 의학적, 과학적 효능을 보장하는 설명이 아니라 오행 상징과 브랜드 경험을 연결하는 콘텐츠로 제공한다. 문구는 “도움이 될 수 있다”, “상징한다”, “추천한다” 수준으로 작성한다.
