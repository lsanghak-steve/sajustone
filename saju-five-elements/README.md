# 오행 사주 계산기

생년월일과 출생시간을 입력하면 사주 기둥과 목, 화, 토, 금, 수 오행 분포를 보여주는 정적 웹앱입니다.

## 실행 방법

브라우저에서 `index.html`을 열면 바로 실행됩니다.

## 현재 계산 범위

- 양력 생년월일 기준
- 년주: 입춘을 2월 4일 기준으로 간단 보정
- 월주: 주요 절기 시작일을 고정 날짜로 간단 보정
- 일주: 율리우스 일수 기반 60갑자 계산
- 시주: 2시간 단위 지지와 일간 기준 시간 천간 계산
- 오행 점수: 각 기둥의 천간과 지지 대표 오행을 1점씩 합산

## 추가된 기능

- 사주 기둥, 일간, 오행 분포, 부족 오행을 바탕으로 약 2000자 내외의 상세 해설을 제공합니다.
- 재물운, 사업운, 연애운, 종합운을 카드 형태로 요약해 보여줍니다.
- 현재 연도의 간지를 자동 계산해 올해 재물운, 사업운, 연애운, 종합운을 별도 카드로 보여줍니다.
- 부족 오행을 기준으로 목, 화, 토, 금, 수 기운에 맞는 천연석을 추천합니다.
- 외부 API 없이 브라우저에서 정적 템플릿으로 생성합니다.
- 추후 전문 만세력 계산 또는 AI 해설 API를 붙일 경우 generateDetailedReading 함수 중심으로 교체할 수 있습니다.

## 고도화 필요 사항

- 실제 절기 시각 기반 계산
- 음력 생일 입력 및 윤달 변환
- 출생지/타임존 보정
- 지장간 가중치 반영
- 부족 오행별 상품 추천 연결
- 상담/마케팅 문구 CMS화
