# 오행 사주 계산기

현재 버전: `1.0.0`

양력 또는 음력 생년월일, 성별, 출생시간을 입력하면 사주 4주, 오행 분포, 상세 만세력 리포트, 사주 해설, 부족 오행 기반 천연석 추천을 보여주는 정적 웹앱입니다.

## 실행 방법

로컬 개발 서버를 실행하면 API 라우트까지 함께 테스트할 수 있습니다.

```powershell
node dev-server.mjs
```

브라우저에서 `http://127.0.0.1:4174/`로 접속합니다.

## 현재 계산 범위

- 양력/음력 생년월일 입력 지원
- 음력 윤달 여부 선택 지원
- 한국천문연구원 음양력 API 기반 양력/음력 변환 및 일진 조회
- 년주: 입춘 기준 간단 보정
- 월주: 주요 절기 시작일 고정 날짜 기준 보정
- 일주: 공식 일진이 있으면 우선 반영, 없으면 율리우스 일수 기반 60갑자 계산
- 시주: 2시간 단위 지지와 일간 기준 시간 천간 계산
- 오행 점수: 각 기둥의 천간과 지지 대표 오행을 1점씩 합산
- 대운 시작 나이: 출생일시와 순행/역행 기준 절기 차이를 월 단위로 환산

## 주요 기능

- 사주 4주와 오행 분포 표시
- 공식 음양력 정보 표시
- 부족 오행 안내
- 올해 운세 카드 제공
- 상세 만세력 리포트 제공
- 약 2000자 내외의 사주 상세 해설 제공
- 부족 오행 기반 천연석 추천
- 원석/팔찌 이미지 표시 및 새 탭 구매 링크 연결
- 원석 전체 보기 페이지 제공
- Vercel 서버리스 API 연동 지원

## 환경변수

`.env`는 Git에 올리지 않습니다. 로컬 개발에서는 루트 `.env`를 사용하고, Vercel에서는 Environment Variables에 아래 값을 등록합니다.

```env
KASI_SERVICE_KEY=공공데이터포털_인증키
KASI_LUNAR_API_ENDPOINT=https://apis.data.go.kr/B090041/openapi/service/LrsrCldInfoService
```

## 고도화 필요 사항

- 실제 절기 시각 기반 월주/년주 정밀 보정
- 출생지/타임존 보정
- 지장간 가중치 반영
- 신살/십성 계산 로직 고도화
- 실제 상품 촬영 이미지로 원석/팔찌 이미지 교체
- 상담/마케팅 문구 CMS화
