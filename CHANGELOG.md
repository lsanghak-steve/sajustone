# Changelog

## 1.0.0 - 2026-05-27

첫 정식 릴리스입니다.

### 주요 기능

- 양력/음력 생년월일, 성별, 출생시간 입력 지원
- 음력 윤달 입력 및 한국천문연구원 API 기반 양력 변환
- 한국천문연구원 음양력 API 기반 공식 음력 정보, 일진, 율리우스 적일 표시
- 공식 일진을 활용한 일주 보정
- 사주 4주와 오행 분포 계산
- 부족 오행 안내
- 올해 운세 카드 제공
- 상세 만세력 리포트 제공
  - 사주 4주
  - 오행 분포
  - 십성/지장간/12운성
  - 대운/세운/月운
  - 천간/지지 관계
  - 신살 요약
- 절기 차이 기반 대운 시작 나이 계산
- 약 2000자 내외 사주 상세 해설 제공
- 부족 오행 기반 천연석 추천
- 원석/팔찌 이미지 표시 및 구매 링크 연결
- 원석 전체 보기 페이지 제공
- Vercel 루트 리다이렉트와 서버리스 API 구성

### 운영 메모

- `.env`는 Git에 포함하지 않습니다.
- Vercel 배포 시 `KASI_SERVICE_KEY`, `KASI_LUNAR_API_ENDPOINT`를 Vercel Environment Variables에 등록해야 합니다.
